# Templates Directory

This folder stores reusable templates.

Use it for:

- project templates,
- document templates,
- planning templates,
- prompt templates,
- reporting templates.

## Registered Templates

### `demo3`

- Type: React dashboard starter template
- Location: `templates/demo3`
- Use it when a task needs a ready-made React admin/dashboard base that should be adapted rather than built from scratch.
- Default approach:
  copy the template into a new working project, then customize branding, pages, components, routes, and content to match the request.

### `anthropic-managed-agents/web-scraper-system-prompt.txt`

- Type: system prompt template
- Location: `templates/anthropic-managed-agents/web-scraper-system-prompt.txt`
- Use it when creating a Browser Use Cloud scraping agent in Anthropic Managed Agents.
- Default approach:
  reuse the prompt as a base, then adjust extraction rules and output fields for the specific scraping task.
