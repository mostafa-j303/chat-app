# React Agent Skill

This skill helps the agent work effectively on React-based projects.

## Use This Skill When

- building React components,
- improving UI structure,
- fixing React bugs,
- organizing frontend code,
- creating reusable interface patterns.

## Preferred Focus

- clear component structure,
- readable props and state flow,
- reusable UI patterns,
- good folder organization,
- maintainable styling choices.

## Workflow

1. Inspect the project structure first.
2. Understand the existing patterns before changing them.
3. Keep components focused and readable.
4. Reuse existing design and code conventions where possible.
5. Document important new patterns if they are reusable.

## Notes

- Prefer practical solutions over over-engineering.
- Match the existing design system when one already exists.
- When new React patterns become stable, record them in this skill or in `MEMORY.md`.
