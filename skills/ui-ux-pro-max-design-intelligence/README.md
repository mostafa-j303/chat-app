# UI UX Pro Max Design Intelligence Skill

This skill helps the agent make stronger UI and UX decisions with a higher design standard than basic screen assembly.

## Use This Skill When

- designing or redesigning product screens,
- improving the quality of layout, hierarchy, and interaction flow,
- turning rough ideas into polished interface direction,
- refining dashboards, admin panels, landing pages, or app experiences,
- choosing patterns that should feel modern, intentional, and premium.

## Preferred Design Direction

- clear hierarchy before decoration,
- polished layouts with strong spacing rhythm,
- purposeful typography and visual contrast,
- interactions that feel smooth, guided, and confident,
- interfaces that balance beauty, clarity, and conversion.

## UX Focus

- identify the main user goal first,
- reduce friction in high-value actions,
- keep navigation and state changes easy to understand,
- use motion, feedback, and structure to guide attention,
- remove clutter that does not help the task.

## Workflow

1. Understand the product goal and user intent.
2. Identify the key screen actions and pain points.
3. Build a stronger hierarchy around the most important action.
4. Improve layout, readability, spacing, and flow before adding flourish.
5. Use animation, components, and visual patterns only when they support clarity.
6. Save reusable high-quality patterns for future work.

## Notes

- Avoid generic AI-looking layouts.
- Prefer premium, thoughtful UI over busy or trendy UI.
- Combine this skill with React, UI Design, or Framer Motion when the project needs implementation as well as design direction.
