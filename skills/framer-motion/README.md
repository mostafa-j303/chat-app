# Framer Motion Skill

This skill helps the agent use Framer Motion in a clean, purposeful way inside React interfaces.

## Use This Skill When

- adding motion to React components,
- creating page transitions,
- animating cards, sections, modals, or menus,
- improving perceived quality with entrance, exit, hover, or staggered motion,
- replacing static UI interactions with intentional animation.

## Preferred Motion Direction

- use motion to clarify hierarchy and state,
- prefer subtle and meaningful animation over constant movement,
- use stagger, fade, slide, scale, and layout transitions where they improve comprehension,
- keep timing consistent across related components,
- avoid noisy or decorative animation that hurts usability.

## Workflow

1. Understand the user action or UI state that needs motion.
2. Choose the smallest animation pattern that improves clarity.
3. Reuse shared timing, easing, and variants when possible.
4. Keep animations lightweight and easy to maintain.
5. Respect the existing design system and interaction style.

## Notes

- Prefer purposeful motion over flashy motion.
- Use Framer Motion to support layout, navigation, and feedback.
- When a motion pattern becomes reusable, save it as a repeatable pattern in this skill or a related template.
