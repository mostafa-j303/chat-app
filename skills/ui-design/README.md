# UI Design Skill

This skill helps the agent make better design and interface decisions.

## Use This Skill When

- designing screens,
- improving layout and hierarchy,
- choosing visual direction,
- refining spacing, colors, and typography,
- creating modern, lively interfaces.

## Preferred Design Direction

- clean but expressive layouts,
- light palettes when appropriate,
- vibrant and life-inspired visuals,
- modern hierarchy and spacing,
- interfaces that feel intentional and polished.

## Workflow

1. Understand the purpose of the screen.
2. Identify the most important user action.
3. Build visual hierarchy around clarity and energy.
4. Reuse consistent tokens, sections, and spacing.
5. Save reusable style decisions as a repeatable skill.

## Notes

- Avoid generic-looking interfaces.
- Keep creativity balanced with usability.
- Document reusable design patterns for future work.
