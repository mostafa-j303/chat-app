# Skills Directory

This folder stores reusable skills, workflows, and capability notes for the AI agent.

Use it for:

- framework-specific instructions,
- tool usage patterns,
- coding workflows,
- design workflows,
- reusable prompt patterns,
- lessons that should become repeatable skills.

Suggested structure:

- one folder per skill, or
- one markdown file per skill.

Example future entries:

- `react/`
- `nextjs/`
- `framer-motion/`
- `anthropic-managed-agents/`
- `ui-ux-pro-max-design-intelligence/`
- `ui-design/`
- `prompt-engineering.md`
